# 时间线图片组查看功能

## 📋 需求说明

**功能**：时间线中的图片按组显示，点击任意一张后可以在组内切换查看，而不是每次只能看一张。

**适用终端**：
- ✅ 用户端 (`index.html`)
- ✅ 司机端 (`driver.html`)
- ✅ 管理端 (`admin.html`)

## 🎨 界面效果

```
时间线：
├─ 15:00 司机已到达现场
│   └─ [图 1] [图 2] [图 3]
│        ↑
│     点击这张
│
▼ 弹窗全屏显示
┌─────────────────────────────────┐
│  ←  [图片 2/3]  →               │
│                                 │
│      ┌─────────────┐            │
│      │             │            │
│      │   大图显示   │            │
│      │             │            │
│      └─────────────┘            │
│                                 │
│  [1] [2●] [3]  ← 缩略图导航     │
│                                 │
│          [× 关闭]                │
└─────────────────────────────────┘
```

## 🔧 技术实现

### 1. 图片组数据结构
```javascript
{
  groupId: 'timeline-1500',  // 时间线索引
  photos: ['/upload/1.jpg', '/upload/2.jpg', '/upload/3.jpg'],
  currentIndex: 1  // 当前显示第 2 张（从 0 开始）
}
```

### 2. 核心功能
- ✅ 点击任意照片打开弹窗
- ✅ 显示当前照片在组内的位置（2/3）
- ✅ 左右箭头切换上一张/下一张
- ✅ 底部缩略图导航
- ✅ 键盘左右键切换
- ✅ 点击背景关闭

### 3. 实现方案
- 复用现有的 `viewPhoto` 函数
- 扩展为 `viewPhotoGroup(groupId, photos, index)`
- 添加全局状态管理当前图片组
- 支持键盘事件
