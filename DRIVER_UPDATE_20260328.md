# 司机端功能更新说明

## 更新时间
2026-03-28

## 更新内容

### 1. 接单状态开关 ✅

**功能描述：**
- 司机可以在任务列表页面顶部开启/关闭接单状态
- 开启接单时，新订单将推送给司机
- 关闭接单时，系统不会派单给司机

**界面位置：**
- 任务列表页面顶部，统计卡片上方

**技术实现：**
- 后端新增接口：`PUT /api/drivers/profile/:id/toggle-accepting`
- 数据库新增字段：
  - `accepting_orders` (INTEGER): 1=开启接单，0=关闭接单
  - `accepting_orders_updated_at` (DATETIME): 状态更新时间
- 前端自动同步状态，带视觉反馈（绿色=开启，灰色=关闭）

### 2. 腾讯地图路线显示 🗺️

**功能描述：**
- 在任务详情页面显示从司机当前位置到救援地点的路线
- 实时计算距离和预计时间
- 支持一键唤起腾讯地图导航

**界面位置：**
- 任务详情页面顶部

**技术实现：**
- 使用腾讯地图 JS API v2
- 后端新增接口：`POST /api/drivers/route`（腾讯地图方向 API 代理）
- 前端功能：
  - 自动获取司机当前位置（GPS 定位）
  - 地图显示起点（司机位置）和终点（救援地点）
  - 绘制行驶路线（青色高亮）
  - 显示距离和预计时间
  - "开始导航"按钮唤起腾讯地图 APP

### 3. 订单列表增强 📋

**功能描述：**
- 订单列表现在可以点击查看详情
- 详情页中可以直接查看路线和导航
- 根据不同任务阶段显示不同操作按钮：
  - 待确认：确认任务 / 拒绝任务
  - 已确认：导航去现场 / 到达现场
  - 已到达现场：导航去目的地 / 到达目的地
  - 已到达目的地：完成任务

## 使用说明

### 司机端操作流程

1. **登录司机端**
   - 访问：`http://localhost:3000/driver.html`
   - 测试账号：13900139001 / 123456

2. **开启/关闭接单**
   - 在任务列表页面顶部切换接单状态开关
   - 绿色表示开启接单，灰色表示关闭接单

3. **查看订单详情**
   - 点击任意订单卡片查看详情
   - 顶部自动显示地图和路线

4. **使用导航**
   - 在任务详情页查看路线信息
   - 点击"开始导航"按钮唤起腾讯地图
   - 或点击"导航去现场"/"导航去目的地"直接开始导航
