## ✅ 测试步骤

### 1️⃣ 测试接单状态开关

1. 登录司机端（使用任意测试账号）
2. 在任务列表页面顶部看到"📡 接单状态"卡片
3. 切换开关：
   - ✅ 开启时显示绿色，文字"接单中"
   - ❌ 关闭时显示灰色，文字"已暂停接单"
4. 切换时会弹出提示消息
5. 刷新页面，状态应该保持

### 2️⃣ 测试地图路线显示

1. 点击任意订单卡片
2. 在详情页顶部看到"🗺️ 导航路线"区域
3. 允许浏览器获取位置权限
4. 地图应该显示：
   - 蓝色标记：你的当前位置
   - 红色标记：救援地点
   - 青色路线：行驶路线
5. 查看路线信息卡片：
   - 距离（例如：5.2km）
   - 预计时间（例如：15 分钟）

### 3️⃣ 测试导航功能

1. 在任务详情页点击"🧭 开始导航"
2. 应该在新窗口打开腾讯地图导航
3. 或者点击"导航去现场"/"导航去目的地"直接开始导航

### 4️⃣ 测试完整任务流程

**场景 1：新任务**
```
点击订单 → 看到"确认任务"和"拒绝任务"按钮
→ 点击"确认任务"
→ 按钮变为"导航去现场"和"到达现场"
```

**场景 2：已确认任务**
```
点击"导航去现场" → 打开腾讯地图导航
→ 点击"到达现场"
→ 按钮变为"导航去目的地"和"到达目的地"
```

**场景 3：已到达现场**
```
点击"导航去目的地" → 打开腾讯地图导航
→ 点击"到达目的地"
→ 输入签收人姓名
→ 按钮变为"完成任务"
```

**场景 4：完成任务**
```
点击"完成任务"
→ 输入拖车费、里程费、其他费用
→ 确认完成
→ 订单状态变为"已完成"
```

## 🐛 常见问题

### Q: 地图不显示？
**A:** 检查以下几点：
- 是否允许了浏览器获取位置权限
- 浏览器控制台是否有错误
- 腾讯地图 API Key 是否有效

### Q: 路线不显示？
**A:** 可能是地址解析失败，当前使用模拟坐标，后续会优化地理编码功能。

### Q: 接单状态切换失败？
**A:** 检查：
- 后端服务是否正常运行
- 数据库是否正确迁移（检查 drivers 表是否有 accepting_orders 字段）
- 浏览器控制台网络请求是否有错误
