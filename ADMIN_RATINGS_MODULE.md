# 后台管理端评价管理功能

## 📋 功能概述

为后台管理系统添加**评价管理模块**，管理员可以查看所有用户评价、筛选评价、查看评价详情，并删除恶意或不实评价。

## 🎯 主要功能

### 1️⃣ 评价列表查看
- ✅ 显示所有用户提交的评价
- ✅ 分页展示（每页 50 条）
- ✅ 显示关键信息：订单号、司机、评价用户、评分、评价内容、评价时间

### 2️⃣ 评价筛选
- ✅ **按司机筛选** - 查看特定司机的所有评价
- ✅ **按评分筛选** - 查看特定星级的评价（1-5 星）
- ✅ **组合筛选** - 同时按司机和评分筛选
- ✅ **重置筛选** - 一键恢复显示所有评价

### 3️⃣ 评价详情
- ✅ 完整评价信息展示
- ✅ 订单信息（订单号、服务地点、目的地）
- ✅ 用户信息（评价用户）
- ✅ 司机信息（被评价司机）
- ✅ 评分详情（星级、评分内容）
- ✅ 评价时间

### 4️⃣ 评价管理
- ✅ **删除评价** - 管理员可删除恶意或不实评价
- ✅ **自动重新计算** - 删除评价后自动更新司机平均评分
- ✅ **二次确认** - 删除前需要确认，防止误操作

## 📊 后端 API

### 新增接口

#### 1. 获取评价列表
```
GET /api/admin/ratings
```

**参数：**
- `page` - 页码（默认 1）
- `limit` - 每页数量（默认 50）
- `driver_id` - 司机 ID（可选，筛选特定司机）
- `rating_filter` - 评分过滤（可选，1-5）

**响应：**
```json
{
```

### 关键代码

#### 删除评价并重新计算司机评分
```javascript
// 删除评价
await run('DELETE FROM order_ratings WHERE id = ?', [ratingId]);

// 重新计算司机平均评分
const driverId = existing.driver_id;
if (driverId) {
  const avgResult = await get(
    'SELECT AVG(user_rating) as avg_rating FROM order_ratings WHERE driver_id = ? AND user_rating IS NOT NULL', 
    [driverId]
  );
  const newRating = avgResult && avgResult.avg_rating 
    ? Math.round(avgResult.avg_rating * 10) / 10 
    : 5.0;
  await run('UPDATE drivers SET rating = ? WHERE id = ?', [newRating, driverId]);
}
```

## ✅ 使用场景

### 场景 1：查看司机评价
1. 管理员登录后台
2. 点击"⭐ 评价管理"
3. 选择特定司机
4. 查看该司机的所有评价

### 场景 2：查看差评
1. 进入评价管理
2. 评分筛选选择"⭐ (1 星)"或"⭐⭐ (2 星)"
3. 查看所有差评
4. 分析差评原因

### 场景 3：删除恶意评价
1. 发现某条评价为恶意差评
2. 点击"查看"查看详情
3. 确认评价内容不实
4. 点击"🗑️ 删除评价"
5. 二次确认后删除
6. 系统自动重新计算司机评分

### 场景 4：查看评价趋势
1. 查看评价列表
2. 观察评分分布
3. 分析服务质量变化

## 📊 数据流转

```
用户评价司机
    ↓
保存到 order_ratings 表
    ↓
更新司机平均评分 (drivers.rating)
    ↓
后台可查看、筛选、管理
    ↓
管理员删除评价（可选）
    ↓
重新计算司机平均评分
```

## 🔒 权限控制

- ✅ 仅管理员可访问评价管理模块
- ✅ 需要登录后台管理系统
- ✅ 删除操作需要二次确认
- ✅ 所有操作记录在案（可通过日志追溯）
